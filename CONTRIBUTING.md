# Contributing to BAM Engine

Thank you for your interest in BAM Engine!

BAM Engine is currently being developed as part of a master's thesis in Computer
Science. During this phase, **external code contributions (pull requests, code
patches) cannot be accepted**.

Even though code contributions are not accepted right now, there are several ways to
participate:

- **Bug reports & feature requests**: found a bug or have an idea? Open an issue on
  the [issue tracker](https://github.com/kganitis/bam-engine/issues).
- **Documentation feedback**: spotted an error, unclear explanation, or missing
  topic? Let me know via an issue.
- **Testing & reporting**: try the framework, run simulations, and report any
  unexpected behavior.
- **Questions & discussions**: ask questions or start a discussion on GitHub Issues.
- **Spread the word**: star the repository or share the project with colleagues.
