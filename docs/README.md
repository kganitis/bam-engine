# BAM Engine Documentation

This directory contains the Sphinx documentation for BAM Engine.

## Building Documentation

```bash
# Install docs dependencies
pip install -e ".[docs]"

# Build HTML documentation
cd docs
sphinx-build -b html . _build/html

# View locally
open _build/html/index.html
```

## Directory Structure

```
docs/
├── conf.py              # Sphinx configuration
├── index.rst            # Main landing page
├── api/                 # API reference (autosummary)
│   └── index.rst
├── user_guide/          # User guide pages
├── auto_examples/       # Generated from examples/ by sphinx-gallery
├── _build/              # Build output (gitignored)
└── _static/             # Static assets
```

## Auto-Generated Files

The following are generated during build and should not be edited manually:

- `auto_examples/` - Generated from `examples/` by sphinx-gallery
- `api/generated/` - Generated by autosummary from docstrings
- `sg_execution_times.rst` - Sphinx-gallery timing report

## Adding Content

- **Examples**: Add `.py` files to `examples/basic/`, `examples/advanced/`, or `examples/extensions/`
- **API docs**: Add docstrings to source code; autosummary generates pages
- **User guide**: Edit `.rst` files in `user_guide/`
