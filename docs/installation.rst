Installation
============

Requirements
------------

BAM Engine requires Python 3.11 or later.

Install from PyPI
-----------------

.. code-block:: bash

   pip install bamengine

Install with pandas support (for DataFrame export):

.. code-block:: bash

   pip install bamengine[pandas]

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/kganitis/bam-engine.git
   cd bam-engine
   pip install -e ".[dev]"
